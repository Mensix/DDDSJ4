# DDDSJ4

The small tool to generate DSJ4 compatible XML files from input .obj and .mtl files.

## Running

1. Clone this repository and unpack it
2. Run ```cmd.exe``` in this directory
3. Run ```dotnet run <file_name.obj> <file_name.mtl>```
